# TwentyTwentyTwoPlus Plugin

Css classes : [class.txt](https://raw.githubusercontent.com/Asfris/TwentyTwentyTwoPlus/main/class.txt)

# Build project

install Dependencies

`npm i`

Build

` npm run build`

Build and watch

`npm start`
