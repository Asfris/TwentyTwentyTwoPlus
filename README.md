# TwentyTwentyTwoPlus Plugin

A better version of wordpress twentytwentytwo theme.

# Build project

install Dependencies

`npm i`

Build

` npm run build`

Build and watch

`npm start`

# Css classes

`.box-br-s` [ Box ] -> Border radius and shadow
